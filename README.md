# TinyCat Library Automation

Automated Python script using Playwright to streamline bulk patron deletion in TinyCat Library Management System.

## Features
- Automated patron search and deletion by bar ID
- Smart modal navigation and form interaction
- Detailed logging and error handling
- Processes 400+ records efficiently

## Technologies
- Python 3.x
- Playwright (Web Automation)
- Chromium Browser Driver

## Installation
```bash
pip install playwright
playwright install chromium
```

## Usage

1. Update the `student_ids` list in `main2.py` with your patron IDs
2. Run the script:
```bash
python main2.py
```
3. Log into TinyCat when prompted
4. Press ENTER to start automation

## Results
Successfully automated the deletion of 400+ patron records, reducing manual processing time from hours to minutes.
